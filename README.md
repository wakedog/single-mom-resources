# Single Mother Resource Guide — Chico, CA & Alliance, OH

A complete, free, single-page guide to every benefit, shelter, food, healthcare, childcare, legal, and crisis resource available to single mothers with young children in Butte County, California and Stark County, Ohio.

**Live site:** https://wakedog.github.io/single-mom-resources/

## What's inside

- Step-by-step action checklists (Phases 1–4)
- Realistic wait-time tables for every major benefit program
- Counselor tips & insider guidance
- "What to actually say" phone-call scripts
- "When things go wrong" troubleshooting
- Printable pocket reference card
- Downloadable phone-contact cards (.vcf)
- Sections for: cash aid, food, housing, healthcare, childcare, mental health, domestic violence, child support, custody, family law, free legal aid, education, transit, baby supplies, diapers, car seats, taxes (EITC/CTC)

## Features

- Works offline once loaded
- Saves checklist progress automatically (localStorage, never leaves your device)
- Dark mode, font-size controls, print-optimized
- Full-text search across all resources
- Click-to-copy phone numbers
- Auto Google Maps links for every address
- Mobile-friendly, screen-reader accessible

## Updating

Edit `index.html`, then:

```bash
git add index.html
git commit -m "Update resources"
git push
```

GitHub Pages auto-deploys the new version within a minute.
